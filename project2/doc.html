Plan

I plan on creating an improved gif finder site using the GIPHY API once again. This is intended for those that want to find gifs from GIPHY, and I intend on adding the functionality of keeping recent searches, favoriting gifs to return to, and the ability to copy the URL of a favorited gif to their clipboard with a button.

Mobile Layout

Special Features

  • Clicking gifs will favorite them
  • Clicking favorited gifs or the same gif from search results will unfavorite them
  • When restarting the site, it saves the last search in local storage and keeps it in the search field for next time
  • There is a loading bar with a purple bar cycling around a gray wheel but it typically finds results too quickly to see for long

Self Grade

I would probably give myself an 85-90 at most. this is mainly due to the fact that despite feeling like I have added quite a bit of functionality to the system through favorites, local storage of previous search terms, the loading bar, and the changed layout, I feel like it is still quite similar to the previous gif finder in some way despite having different nearly everything except for main function. As I am not the best designer, I also wouldn't give the design that high of a score, hence 85-90.
