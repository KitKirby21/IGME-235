Game Name by Kirby Saligo

High Concept

It's the snake game, except you're in space and trying to eat something too big for you will kill you just as much as running into yourself!


Genre

It's hard to put a genre on the snake game but the closest I can think of would be an avoidance growth simulation I guess.


Platform

Desktop only


Story

This is generally abstract; with the basic premise of a snake that wants to grow BEEG and eats increasingly larger celestial objects in order to achieve this.


Aesthetics

This would be a relatively cartoonish graphic, with maybe a looped noise and a crunch sound effect with everything eaten.


Gameplay

The main game mechanic would be changing the direction of the heading of the snake in order to avoid obstacles or acquire sustenance.

Documentation

Process

My process for this project was small little goals each day. For example, the first day one of my goals was to set up the movement of the singular snake head. The day after, I fixed bugs in its movement such as it being able to go backwards when it should just be able to turn. After that, I implemented food and the ability to grow in length, followed by the ability to die and restart. Only after the game was fully functional did I add auxiliary parts such as a game screen, a score tracker, and high score local storage.

Anything Special

The most special things that I did for this project was creating changing the way that food appears between six different images through java script. Aside from that, I also implemented a local storage for the high score. In terms of project requirements, I met the requirements by having it be an easy simple Snake game. The window adjusts to the screen size of the user, and images are stored properly. All HTML/CSS is valid, and I use an ES6 class within the code which is commented well.
